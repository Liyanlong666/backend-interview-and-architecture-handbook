# JVM 面试核心

Java 虚拟机是 Java 程序的运行基础，也是面试中的重难点。

## 目录
- [[1-JVM基础]] - 内存结构、对象创建、内存溢出
- [[2-垃圾收集]] - 垃圾回收算法、垃圾收集器 (CMS, G1, ZGC)
- [[3-类加载机制]] - 类加载过程、双亲委派模型
- [[4-JVM调优]] - 监控工具、参数配置、调优实战
- [[5-JVM牛客面经八股]] - 高频面试原题汇总

## 高频考点
- 内存区域：堆, 栈, 方法区, 程序计数器
- 垃圾回收：分代收集理论、标记-清除/复制/整理算法
- 类加载：加载, 验证, 准备, 解析, 初始化
- 调优工具：jstack, jmap, jstat, VisualVM, Arthas
