## 高频考点
- 事务四大特性 ACID：原子性、一致性、隔离性、持久性
- 隔离级别：读未提交、读已提交、可重复读（默认）、串行化
- 脏读/不可重复读/幻读：脏读读到未提交，不可重复读两次结果不同，幻读多出或少了行
- undo log：实现事务回滚，保证原子性，MVCC 实现
- redo log：实现持久性，写入快（顺序写），crash-safe
- binlog：归档日志，主从复制，数据恢复
- MVCC：多版本并发控制，通过 undo log 实现，解决幻读
